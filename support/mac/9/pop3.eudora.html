Eudora 5 for MacOS


Select "Settings" from the "Special" menu.

Click on the "Getting Started" icon on the left side of the Settings box and set it up like so:

Username me@example.com
Mail Server mail.example.com
Real Name John Doe
SMTP Server mail.example.com
Return Address me@example.com

 

Then click the "Checking Mail icon and set it up like so:

Username: me@mydomain.com
Mail Host mail.example.com
Mail Protocol POP          IMAP

You're all set! Close up the Settings window and check your mail.

 

NOTE: If you set it up like suggested, you MUST check your email before you can send mail through our smtp server.
